About AICREATIONS

Designed for daily updates without technical chaos.

This website is structured to work smoothly on GitHub and Netlify. You can update content by replacing video and image files inside the assets folder, then pushing the changes to GitHub. Netlify will redeploy the site automatically.

Easy Update Flow
  • Replace the two video files inside assets/videos.
  • Replace the four image files inside assets/images.
  • Commit and push to GitHub.
  • Netlify publishes the update automatically.
